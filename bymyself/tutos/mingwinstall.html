como instalar un compilador de C en tu computadora?

este es un breve tutorial para instalar un compilador de C en tu computadora windows 11 y usarlo para compilar un hola mundo en bash


índice

  1. descargas
  2. instalación
  3. set up
  4. modo de uso

Descargas

para comenzar, debes descargar MinGW-w64 desde MSYS2 haciendo clic aquí


Instalación

una vez descargado, ejecútalo
sigue los pasos para instalarlo en tu dispositivo


set up

ya que el programa haya terminado de instalarse, se abrirá una terminal de MinGW


en esta parte, debes escribir el siguiente comando y presiona enter:

pacman -S --needed base-devel mingw-w64-ucrt-x86_64-toolchain

seguido de ello, aparecerá un mensaje con todos los paquetes que se intalarán


presiona enter.
después, aparecerá una lista detallada de los archivos que se intalarán y el espacio que ocuparán en tu disco. para prodecer, escribe Y (debes estar conectado a internet para realizar este paso)
los archivos deberían comenzar a descargarse.
una vez terminado este proceso, sal de la terminal y busca "editar las variables de entorno de esta cuenta"

en esta ventana, selecciona "Path" y después "editar"

en esta parte, selecciona "Nuevo" y "Examinar"
deberás buscar la carpeta C:\msys64\ucrt64\bin
pulsa "Aceptar" hasta salir de la configuración

para confirmar que se instaló correctamente, en una nueva terminal escribe el commando "gcc --version"



modo de uso

ahora que tenemos MinGW instalado, podemos compilar códigos de C. para hacer un hola mundo, abre una nueva terminal, y en la carpeta de tu preferencia escribe "vi helloworld.c" y copia el siguiente código.

para salir de la edición del código, pulsa ESC y escribe :x
ahora escribe el comando "gcc helloworld.c"
esto creará un nuevo archivo ejecutable, llamado "a.exe"
para ejecutar este archivo, escribe "./a.exe"
si lo hiciste bien, se debería mostrar el mensaje de "hello acm world!"

ahora puedes comenzar a crear y compilar tus programas de C!
